
# Financial Market Analysis: Suzlon Energy (2021–2026) 📈

## 📌 Project Overview
This project presents a financial market analysis of Suzlon Energy Limited using an interactive Power BI dashboard. The analysis is based on five years of historical stock market data obtained from the National Stock Exchange (NSE).

The raw dataset was available in CSV format and was cleaned, transformed, and analyzed in Power BI to visualize price movements, trading volumes, and overall market performance.

## 📁 Dataset Description
The dataset consists of historical stock market data for Suzlon Energy covering **February 2021 to February 2026**.

**Source:** NSE India  
**File Format:** CSV

A total of **5 CSV files (one for each year)** were imported into Power BI and combined to create a single dataset for analysis.

**Key Columns**
- Date  
- Open  
- High  
- Low  
- Close  
- VWAP  
- 52 Week High / Low  
- Volume  
- Number of Trades

## 🧠 Technical Workflow

**Data Preparation**
- Imported **5 CSV files** into Power BI
- Cleaned and transformed data using Power Query
- Standardized date formats and removed inconsistencies
- Merged the datasets to create a unified timeline

**Data Modeling**
- Created calculated measures using DAX
- Built time-based analysis for yearly and monthly trends

**Dashboard Design**
- Designed an interactive financial dashboard
- Implemented slicers for year and month filtering
- Used charts and KPIs for clear visualization
